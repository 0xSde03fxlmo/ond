Installation Instructions for Apache 2 Server

To correctly install your certificate, it is important to configure the server to use the intermediate CA_Bundle.crt file in addition to www_example_com.crt.

Our online Apache installation guide is available here: https://www.geocerts.com/support/install-ssl-certificate-apache-web-server

After installing your certificate, you can verify that it is correctly installed by using our SSL Installation Checker tool: https://www.geocerts.com/ssl-checker

If you are having difficulty installing your certificate, please contact our friendly support team for more assistance at https://www.geocerts.com/support.
